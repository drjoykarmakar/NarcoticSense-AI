# Chemist Quickstart

Open the app with `START_HERE.command`.

## Recommended workflow

1. Import one or more spectra.
2. Check the Dataset tab to verify x/y ranges.
3. Use Spectra Viewer for raw/processed overlays.
4. Use Peak Analysis for peak positions, heights, areas, and metrics.
5. Upload multiple spectra and use Chemometrics for PCA and clustering.
6. Export metadata templates and aligned matrices.
7. Download a Markdown report.

## Good data practice

Record compound/class, concentration, solvent, pH, temperature, instrument, integration time, operator, date, and notes for every spectrum.
