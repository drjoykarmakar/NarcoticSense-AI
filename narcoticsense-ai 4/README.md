# NarcoticSense AI

Chemist-friendly AI spectroscopy operating system for research and decision support.

## Start on Mac

1. Unzip the folder.
2. Double-click `START_HERE.command`.
3. Upload spectroscopy CSV/TXT files.

This beginner build does **not** require `pip install -e .`.

## What v4 includes

- Universal spectroscopy importer for common CSV/TXT vendor-like exports
- Multi-file upload
- Dataset manager and metadata template export
- Raw, processed, overlay, and derivative spectra viewer
- Peak table and spectral metrics
- PCA, t-SNE, and K-means chemometrics
- Aligned spectral matrix export for future ML
- No-code AI dataset planning tab
- Markdown research report generator

## Responsible use

This software is for research, screening, visualization, and decision support. Confirm high-stakes results using validated laboratory methods.
