# NarcoticSense AI Roadmap

## v0.1.0: Stable Chemist Workbench

- Universal CSV importer with automatic x/y detection.
- Multi-file upload.
- Spectral overlays.
- Preprocessing controls.
- Peak metrics and peak table.
- PCA, t-SNE, and K-means exploratory analysis.
- Research report export.
- Streamlit app, tests, Docker, and CI.

## v0.2.0: Professional Spectroscopy Tools

- Excel and TXT importers.
- JCAMP-DX importer.
- More robust vendor format detection.
- Peak integration and FWHM.
- Publication-quality figure export.
- Dataset metadata editor.
- Project save/load.

## v0.3.0: Chemometrics

- PLS and PLS-DA.
- LDA and SIMCA.
- Hierarchical clustering.
- Cross-validation workflows.
- Calibration plots and residual analysis.

## v0.4.0: AI and Uncertainty

- Random Forest, SVM, XGBoost, LightGBM.
- Train/test splitting and validation reports.
- Conformal prediction.
- Out-of-distribution detection.
- Model registry.

## v0.5.0: Multimodal AI

- Raman + fluorescence + IR + UV-Vis fusion.
- Metadata-aware models.
- Molecular descriptor integration.
- Transfer learning across instruments.

## v1.0.0: Research Release

- Stable public API.
- Documentation site.
- Reproducible benchmark datasets.
- Example publication workflow.
- Responsible-use documentation.
