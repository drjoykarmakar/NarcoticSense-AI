# Data and Model Licensing

The source code in this repository is licensed under the Apache License 2.0.

Recommended licensing policy for project assets:

- Original code: Apache License 2.0.
- Documentation: Apache License 2.0 unless otherwise stated.
- Original example datasets: CC BY 4.0 is recommended when you want broad reuse with attribution.
- Pretrained models trained only on permissively licensed or original data: Apache License 2.0 is recommended.
- Third-party datasets and spectral libraries: keep their original license terms and do not redistribute restricted data.

Do not upload proprietary or restricted spectral libraries unless you have explicit permission.
